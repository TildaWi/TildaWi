## 👋 About Me

### 사용자의 여정을 따라가 보는 것을 좋아하는 데이터 분석가, 위이태인입니다.  
- 저는 사용자 행동 데이터를 분석해 제품의 **핵심 이벤트**를 찾아내고, 의사결정을 지원하는 인사이트를 도출하는 데 집중합니다.
- 데이터 속 패턴을 발견해 **제품과 비즈니스가 함께 성장**할 수 있는 연결고리를 만들고, 그 여정에서 저도 함께 성장하는 것이 목표입니다.
